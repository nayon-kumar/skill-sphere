# 📚 SkillSphere - Online Learning Platform

SkillSphere is a modern **E-learning web application** built with Next.js that allows users to explore courses, view details, enroll in courses, and manage their learning journey.

---

## 🚀 Live Demo

👉 https://skill-sphere-pi-inky.vercel.app/

---

## 🧑‍💻 Technologies Used

- **Frontend Framework:** Next.js (App Router)
- **Language:** JavaScript (ES6+)
- **Styling:** Tailwind CSS
- **UI Enhancements:**
  - Framer Motion (animations)
  - Animate.css
- **Authentication:** Better Auth
- **Database:** MongoDB
- **State Management:** React Context API
- **Form Handling:** React Hook Form
- **Notifications:** React Toastify
- **Icons:** React Icons
- **Image Optimization:** Next.js Image Component

---

## ✨ Features

### 🔐 Authentication System

- Email & Password Login/Register
- Google Social Login
- Secure session management
- Protected routes (Profile & Course Details)

### 📚 Course Management

- Browse all courses
- View course details dynamically
- Search courses by title
- View popular and new release courses

### 🎯 Enrollment System

- Enroll in courses
- Prevent duplicate enrollment
- Unenroll functionality
- View enrolled courses in profile

### 👤 User Profile

- View user information
- Edit profile (name & image)
- Display enrolled courses

### 🎨 UI/UX Features

- Fully responsive design
- Smooth animations (Framer Motion)
- Scroll progress indicator
- Video hero section
- Modern card-based UI

### ⚡ Performance & Optimization

- Server-side data fetching
- Dynamic metadata for SEO
- Optimized images using Next.js Image
- Loading and error handling pages

### 🧭 Navigation

- Responsive navbar
- Mobile-friendly menu
- Active route highlighting
